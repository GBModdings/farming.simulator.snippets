## Install

1. Go to Releases [VS Marketplace](https://github.com/GBModdings/farming.simulator.snippets/releases).
2. Download a last vsix package of the extension.
3. Go to VS Code, press the CTRL+SHIFT+X -> ... install from VSIX...
